# lab8-Cross-platform

How to start:

run these commands:

npx create-expo-app NavigationAppFinalWorking --template blank
cd NavigationAppFinalWorking

install these dependencies:

npm install @react-navigation/native @react-navigation/stack @react-navigation/bottom-tabs

place my src folder into the root. 

replace the defaults or place my index.js, app.js, app.json and package.json in the project root. 

run npx expo start
